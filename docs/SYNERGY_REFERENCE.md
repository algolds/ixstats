## Tips for Optimal Government Design

### High Effectiveness Strategies
1. **Choose complementary components** from different categories
2. **Avoid mixing opposites** (e.g., democratic + autocratic)
3. **Stack synergies** by selecting components that share multiple connections
4. **Balance ideology** - pure systems work better than mixed

### Warning Combinations
⚠️ These will ALWAYS conflict:
- Democratic + Autocratic processes
- Free Market + Planned Economy
- Federal + Unitary systems
- Rule of Law + Military Enforcement
- Professional + Partisan institutions

### Powerhouse Combinations
✅ These create strong synergies:
- Democratic + Electoral + Rule of Law + Independent Judiciary
- Autocratic + Centralized + Charismatic + Surveillance
- Technocratic + Performance + Professional + Agencies
- Traditional + Religious + Social Pressure + Consensus

## Implementation Details

**File**: `/src/lib/government-synergy.ts`
**Function**: `checkComponentSynergy(type1, type2)`
**Returns**: `{ type: 'ADDITIVE'|'CONFLICTING', multiplier: 1.0, description: string } | null`

**Calculation**:
```typescript
Base Effectiveness = Σ(component.effectivenessScore) / count
Synergy Bonus = count(ADDITIVE) × 10
Conflict Penalty = count(CONFLICTING) × 15
Final = clamp(Base + Synergy - Penalty, 0, 100)
```
